# 04-layout-system.md  
Universal Layout System for Brand Pages v1.0

---

## 1. 이 문서의 목적

이 문서는 **Brand Page를 구성하는 레이아웃의 ‘종류’와 ‘선택 규칙’을 정의**한다.  
디자인 스타일이나 미적 판단이 아니라,  
**어떤 인식 상태에 어떤 레이아웃이 필요한지**를 규정한다.

이 문서는:
- UI 디자인 가이드 ❌
- 컴포넌트 라이브러리 ❌  
→ **Brand Page를 망치지 않는 화면 구조 규칙**이다.

---

## 2. 레이아웃의 전제

레이아웃은 정보를 담는 그릇이 아니라  
**인식을 조절하는 장치**다.

같은 콘텐츠라도  
레이아웃이 바뀌면 **전혀 다른 브랜드**로 인식된다.
