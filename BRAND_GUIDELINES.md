## 🎯 브랜드 아이덴티티

### 브랜드 네임
```
공식 명칭: KOLLAB KOREA
영문 표기: KOLLAB (항상 대문자)
한글 표기: 콜랩 코리아
```

### 브랜드 슬로건
```
메인: WHERE SEOUL MEETS LA
서브: From Seoul To LA
     K-브랜드의 글로벌 첫걸음
```

### 브랜드 비전
> K-브랜드의 미국 시장 진출을 가장 전략적으로 시작할 수 있는 첫 단계

### 브랜드 미션
- Seoul과 LA를 연결하는 프리미엄 셀렉트 스토어
- K-브랜드의 글로벌 확장 파트너
- 큐레이션된 브랜드 경험 제공

### 브랜드 가치
1. **Bold & Industrial**: 강렬하고 인더스트리얼한 감성
2. **Premium Selection**: 엄선된 브랜드만 입점
3. **Global Vision**: 세계를 향한 확장
4. **Partnership**: 브랜드와 함께 성장

## 🎨 로고 사용 가이드

### 로고 버전

#### 1. Primary Logo (한 줄)
```
KOLLAB
```
- **사용처**: Navbar, Footer, 일반 마케팅 자료
- **장점**: 가독성이 높고 공간 효율적
- **최소 크기**: 60px (높이)

#### 2. Stacked Logo (두 줄)
```
KOL
LAB
```
- **사용처**: Hero, 정사각형 레이아웃, SNS 프로필
- **장점**: 임팩트 있고 기억에 남음
- **최소 크기**: 80px (높이)

### 로고 컬러 버전

| 버전 | 파일명 | 배경색 | 사용처 |
|------|--------|--------|--------|
| **블랙** | `kollab-black.svg` | 화이트/베이지 | About, Benefits 섹션 |
| **화이트** | `kollab-white.svg` | 블랙 | Navbar, Footer, Hero |
| **레드** | `kollab-red.svg` | 화이트 | 특별한 강조가 필요할 때 |

### 로고 사용 규칙

#### ✅ DO (해야 할 것)

```
✅ 정의된 컬러 버전만 사용
✅ 충분한 여백 확보 (최소 로고 높이의 50%)
✅ 배경과 대비가 명확한 버전 선택
✅ 고해상도 버전 사용 (SVG 권장)
✅ 최소 크기 준수
```

#### ❌ DON'T (하지 말아야 할 것)

```
❌ 로고 왜곡하거나 비율 변경
❌ 로고 회전
❌ 로고에 그림자나 효과 추가
❌ 정의되지 않은 컬러 사용
❌ 배경과 대비가 낮은 조합
❌ 다른 요소와 너무 가깝게 배치
```

### 로고 배치 가이드

#### Clear Space (여백)
```
로고 주변에 최소 X만큼의 여백 확보
X = 로고 높이의 50%

예시:
로고 높이 = 100px
최소 여백 = 50px (상하좌우)
```

#### 최소 크기
```
Digital:
- 한 줄 로고: 60px (높이)
- 두 줄 로고: 80px (높이)

Print:
- 한 줄 로고: 15mm (높이)
- 두 줄 로고: 20mm (높이)
```

## 📝 타이포그래피

### Primary Font

```
Font Family: Inter
Designer: Rasmus Andersson
Type: Sans-serif
License: Open Font License

다운로드:
https://fonts.google.com/specimen/Inter
```

### Font Weights

| Weight | 값 | 사용처 |
|--------|-----|--------|
| **Light** | 300 | 보조 텍스트 |
| **Regular** | 400 | Body 텍스트 |
| **Medium** | 500 | Body 강조 |
| **Semi Bold** | 600 | 부제목 |
| **Bold** | 700 | 제목 |
| **Black** | 900 | 대형 제목, CTA |

### 타이포그래피 원칙

1. **대형 제목**: Inter Black (900) + UPPERCASE + Tight Tracking
2. **본문**: Inter Medium (600) + 적절한 Line Height
3. **CTA**: Inter Black (900) + UPPERCASE + Wide Tracking
4. **한글**: Word-break: keep-all 필수

## 🗣️ 톤 앤 매너

### 브랜드 보이스

#### 특징
- **Bold**: 자신감 있고 명확한 메시지
- **Professional**: 전문적이고 신뢰할 수 있는
- **Aspirational**: 글로벌 진출의 꿈을 자극하는
- **Direct**: 직설적이고 간결한

#### 예시

##### ✅ GOOD
```
"WHERE SEOUL MEETS LA"
→ 간결하고 강렬한 메시지

"KOLLAB는 K-브랜드의 미국 시장 진출을 가장 전략적으로 시작할 수 있는 첫 단계입니다."
→ 명확한 가치 제안

"Ready To Go Global?"
→ 직접적인 행동 유도
```

##### ❌ BAD
```
"우리와 함께 하시겠어요?"
→ 너무 부드럽고 약한 메시지

"아마도 좋은 기회가 될 수 있을 것 같습니다"
→ 불확실하고 자신감 부족
```

### 메시지 가이드라인

#### 제목/헤드라인
```
- 짧고 강렬하게 (5-7 단어)
- UPPERCASE 사용
- 명령형 또는 질문형
- 숫자 활용

예시:
- WHERE SEOUL MEETS LA
- READY TO GO GLOBAL?
- 30 BRANDS. ONE PLATFORM.
```

#### 본문
```
- 명확하고 직접적으로
- 혜택과 가치 중심
- 구체적인 숫자 활용
- 행동 유도

예시:
"총 30개 브랜드 입점 예정. 교차 구매 및 체류시간 증가로 매출 상승."
```

#### CTA (Call-to-Action)
```
- 명령형 동사 사용
- UPPERCASE
- 긴급성/독점성 강조

예시:
- APPLY NOW
- JOIN THE MOVEMENT
- GET STARTED
- LIMITED SPOTS
```

## 📸 이미지 가이드라인

### 사진 스타일

#### 특징
- **인더스트리얼**: 콘크리트, 메탈, 노출 구조
- **미니멀**: 깔끔하고 정돈된
- **고대비**: 명암이 뚜렷한
- **그레이스케일**: 컬러 사용 최소화

#### 필터
```
- Grayscale 또는 Desaturated
- 높은 Contrast
- 날카로운 Focus
- 자연광 선호
```

### 제품 사진
```
- 화이트 또는 블랙 배경
- 정면 또는 45도 각도
- 그림자 최소화
- 고해상도 (최소 2000px)
```
