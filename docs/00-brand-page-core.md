# 00-brand-page-core.md  
Universal Brand Page Core Specification v1.0

---

## 1. 이 문서의 목적

이 문서는 **브랜드 페이지 제작의 최상위 기준(Core)** 이다.  
특정 브랜드(KOLLAB 포함)에 종속되지 않으며,  
모든 브랜드 페이지 제작 시 **반드시 선행·참조되어야 하는 절대 규칙**을 정의한다.

이 문서는:
- 디자인 가이드 ❌  
- 스타일 레퍼런스 ❌  
- 콘텐츠 초안 ❌  

→ **“무엇을 만들 것인가”가 아니라  
“어떤 페이지를 만들고 있는가”를 규정한다.**

---

## 2. Brand Page의 정의

### 2.1 Brand Page란 무엇인가

Brand Page는 브랜드의:
- 세계관
- 존재 이유
- 태도
- 축적된 신뢰

를 **시간 순서가 아닌 ‘인식 구조’로 전달하는 페이지**이다.

성과를 증명하지 않는다.  
설득하지 않는다.  
대신 **이해되도록 배치한다.**

---

## 3. Brand Page가 아닌 것 (중요)

아래 중 하나라도 주 목적이면 **Brand Page가 아니다.**

- 제품 판매
- 캠페인 전환
- 즉각적 문의 유도
- 포트폴리오 나열
- 회사 소개서 대체

> Brand Page는  
> “지금 연락하게 만드는 페이지”가 아니라  
> “나중에 떠올리게 만드는 페이지”다.
